# IDS_CIC-IOT-Dataset-2025

Projeto em Python desenvolvido em Jupyter Notebook para um Sistema de Detecção de Intrusão (IDS) utilizando o dataset **CIC-IOT-Dataset-2025**.

## Informações sobre o dataset

- O dataset está disponível no seguinte link:  
  [https://www.unb.ca/cic/datasets/iiot-dataset-2025.html](https://www.unb.ca/cic/datasets/iiot-dataset-2025.html)

## Arquivos principais

- **leitura_arquivo.html**: contém o código para a união dos datasets.
- **codigos_versao_entregavel.html**: contém os códigos utilizados no estudo comparativo.

## Estudo comparativo

O estudo foi realizado comparando três modelos supervisionados de aprendizado de máquina:

- Random Forest
- Gradient Boosting
- Regressão Logística
