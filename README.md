# IDS_CIC-IOT-Dataset-2025
Código python em jupyter notebook para um IDS utilizando o dataset CIC-IOT-Dataset-2025.
O dataset está disponível em: https://www.unb.ca/cic/datasets/iiot-dataset-2025.html
O arquivo HTML "leitura_arquivo" contém o código de união dos dataset
o arquivo HTML "codigos_versao_entregavel" contem os códigos utilizados no estudo comparativo
O estudo foi realizado comparando três modelos supervisionados: Random Forest, Gradient Boosting e Regressão Logística.
